# Tech Document Command

Creates or updates technical documentation in the `tech-docs/` folder.

## Usage

```
/tech-document <what to document>
```

## Instructions

Use the **tech-documenter** subagent to analyze the codebase and create developer-focused documentation.

## Examples

```
/tech-document the entire project
/tech-document API endpoints
/tech-document authentication architecture
/tech-document database schema
```
