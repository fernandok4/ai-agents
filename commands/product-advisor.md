# Product Advisor Command

Ask questions about the product or brainstorm features.

## Usage

```
/product-advisor <question or topic>
```

## Instructions

Use the **product-advisor** subagent to answer product questions, explore feature ideas, or analyze requirements.

## Examples

```
/product How does the payment flow work?
/product Brainstorm ideas for improving onboarding
/product Define requirements for a notification system
/product What features should we prioritize next?
```
